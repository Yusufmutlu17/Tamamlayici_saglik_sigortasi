# 🏥 Sağlık Sigortası Teklif Sistemi 

Bu proje, kullanıcıların en uygun sağlık sigortası tekliflerini anlık olarak hesaplayan ve sunan dinamik bir web uygulamasıdır. Sigorta süreçlerini dijitalleştirerek hem kullanıcılar hem de acenteler için hızlı ve şeffaf bir deneyim sunmayı amaçlar.

## 🚀 Öne Çıkan Özellikler
- **Dinamik Teklif Hesaplama:** Kullanıcının girdiği yaş, kronik hastalık geçmişi ve kişisel alışkanlıklara göre özelleştirilmiş fiyatlandırma algoritması.
- **Kullanıcı Dostu Arayüz:** Form adımlarının kolayca takip edilebildiği, responsive (mobil uyumlu) modern tasarım.
- **Teklif Karşılaştırma & Filtreleme:** Farklı sigorta paketlerinin (Yatarak, Ayakta, Tamamlayıcı vb.) kapsam ve fiyat bazlı karşılaştırması.


## 🛠️ Kullanılan Teknolojiler
C# / ASP.NET Core
